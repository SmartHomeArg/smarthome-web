# Conocimiento privado Smarthome (plantilla)

## Identidad comercial

- Empresa: Smarthome
- Propuesta de valor: seguridad con monitoreo y gestion remota por app
- Segmentos: hogares y comercios

## Kits

### Kit Cam+
- Incluye: carteleria, camara Wi-Fi, tarjeta de memoria, gabinete estanco, jaula de proteccion
- Nota: no incluye sistema de alarma

### Kit Smart 1.1
- Incluye: central de alarma, fuente, gabinete, bateria, llavero, sensor movimiento, sensor apertura, sirena interior

### Kit Smart 2.2
- Base similar a Smart 1.1
- Diferencia: mas sensores de apertura y movimiento

### Kit Smart Cam 2.2
- Base: Smart 2.2
- Extra: suma camara y elementos de Cam+

### Kit Industrial
- Incluye base de alarma + sirena exterior + sensores exteriores cableados doble tecnologia

## Planes

- Plan Video -> asociado a Kit Cam+
- Plan Basic -> asociado a Kit Smart 1.1
- Plan Plus -> asociado a Kit Smart 2.2
- Plan Pro -> asociado a Kit Smart Cam 2.2
- Plan Comercial -> asociado a Kit Industrial

## Prestaciones relevantes

- Monitoreo 24/7 en planes con sistema de alarma
- Gestion por app movil (activar/desactivar, estado de zonas, historial)
- Notificaciones y alertas de eventos
- En planes con camara: funciones de video y grabacion segun plan vigente

## Preguntas frecuentes (FAQ)

### El kit de camara incluye monitoreo de alarma?
No. El Kit Cam+ es de video y no reemplaza un sistema de alarma.

### Se puede controlar desde el celular?
Si, los sistemas compatibles se gestionan desde app movil.

### Los planes se pueden mezclar entre kits?
No. Cada plan esta pensado para su kit asociado.
