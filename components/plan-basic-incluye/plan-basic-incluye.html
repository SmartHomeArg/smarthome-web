Que incluye este plan?

Monitoreo 24/7 de las senales de tu sistema de alarma con operadores listos para actuar ante cualquier evento.

Arma y desarma tu sistema de alarma desde la app en tu celular o tablet, estés donde estés.

Recibe alertas y notificaciones en tiempo real sobre el estado del sistema y cada evento registrado.

Personaliza codigos de usuario para saber quien activa o desactiva el sistema en cada momento.

Boton de panico, emergencia medica y bomberos disponible desde la app y el llavero de control.

Comunicacion multivia Wi-Fi y red de datos 4G para mantener la conexion activa de forma redundante.

Historial de eventos para consultar todo lo que sucedio en tu sistema desde la app.

Mantenimiento y diagnostico remoto para optimizar el funcionamiento sin necesidad de visita tecnica.
